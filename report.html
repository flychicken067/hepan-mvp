CLASSICAL ANNOTATION · 古典详读

合而后分。
分而后明。

An evidence-based reading drawn from Qiongtong Baojian, Bazi Tiyao, and Sanming Tonghui.

合婚一事，古书称之为「鸾凤配」。鸾凤者，非鸳鸯之轻盈，而是高山之上对鸣的灵鸟——彼此的存在，正好是对方所缺的那一笔。本文据二命五行强弱、用神所在、流年大运、八卦卦象，逐项印证。所引典籍均为公开版本，案例对应均有原文出处。

正财 · 伤官

日主丙火，《滴天髓》所谓「丙火猛烈，欺霜侮雪」是也。然此命月令未土秉政，土旺泄火，是格局成立的关键。

「六月丙火，火气渐退，三伏生寒，己土泄气，取壬为用，庚金为佐。庚壬两透，贴身相生，可云科甲名宦。若无庚贴，有壬不见戊出，秀才小富。」 — 《穷通宝鉴》六月丙火

用神判断

本命壬水（七杀）藏于申中，未透天干，按《穷通宝鉴》对照：「有壬不见戊出，秀才小富」一类——格局有，秀气在，但未透不显，需大运送水。

当前大运

三十二至四十二岁行辛卯运（2023–2033）。辛金正财透干，卯木正印生身。表面财来印旺，内里却暗藏「财克印」之冲——也就是，钱多了反而扰心。这十年宜「在赚钱的同时坚持读书写作」，否则财来印散，得不偿失。

戊为大山，亥月寒水秉令，金水极旺（各 7.6 分）。最大缺口在火（仅 0.8 分），《穷通宝鉴》言：

「十月戊土，气薄力微，时值小阳，先用甲木疏土，次取丙火暖土。非甲土不灵，非丙土不暖，安能发生万物。」 — 《穷通宝鉴》十月戊土

同格历史案例

《穷通宝鉴》P.149 录有近代上海实业家王一亭命，其年月日柱与女方完全相同：

「壬申、辛亥、戊子、丁巳。亥宫壬水得禄，财通门户，更有辛金相生。戊土日元……身旺任财，早年西方之运，以身发财。中年之后，一派南方印运，以慈善事业，斐声社会，寿七十二。」

王一亭中年后行南方印运（丙午运）方成大业。女方四十二至五十二岁正进入丙午运——这是历史上同格人最高光的十年。

03

合盘 · 互为用神

+

合盘看的是两人五行的协奏，不是单方面分数。本案核心机制有三层：

一层 · 五行镜像

男方五行强弱排序为：土十 / 木 8.6 / 火 6.4 / 金 6.1 / 水 0.8。
女方五行强弱排序为：金 7.6 / 水 7.6 / 土 5.4 / 木 3.6 / 火 0.8。

男最缺水，女水极旺；女最缺火，男火透干——彼此的最大缺口正好是对方的最大供给。这种镜像关系在合婚里被称作「互为用神」，是诸多组合中最理想的一种。

二层 · 天干六合

男方日干丙火 × 女方月干辛金 →「丙辛合化水」。化合的产物正好是男方命中最缺的壬水用神。古书称这种合为「天作之合」。

三层 · 卦象相生

男方丙火日主对应离卦（☲），女方戊土日主对应艮卦（☶）。上艮下离合成「山火贲」卦：

「贲，亨，小利有攸往。」 — 《周易》贲卦

贲者，文饰也。男方丙火照耀女方戊土的山峦，女方艮土稳稳承载男方的火光，相互装饰，相互成就。这是合婚中难得的相生卦。

04

隐忧 · 寅申冲与化解

+

女方两个申金（年支、日支）同时冲击时支寅木。寅是七杀，对女命而言即夫星。冲克之象，按古书原话，主「夫妻宫动荡」「感情多波折」「夫君易受外力干扰」。

但本命有救：

救法一 · 寅中藏丙：寅木的本气虽是甲木，藏干却含丙火、戊土。寅中之丙，正是女方最缺的用神，也正是男方的日主——也就是说，那个化解她「夫妻宫冲」的丙火，恰恰是男方本人。

救法二 · 大运补火：女方丁未运（2024–2034）丁火是丙之偶，开始暖土；丙午运（2034 后）丙火透干，彻底化解。

救法三 · 男方扶持：男方丙火常年照耀，相当于把寅中之丙引到天干，作用力放大数倍。两人在一起，时间越长越稳。

05

大师娓娓道来

+

「朋友，我把你们俩的命盘看了一遍，有意思。」

「你是太阳，她是大山。太阳本来就是为了照山的——山没有阳光就是荒地，太阳没有山就是空照。这话不是我编的，《周易》里写得明明白白。」

「她现在最重要的十年才刚开始。你别拿你 22–32 岁那段壬辰运的爆发感来对照她现在——她的爆发期还在 2034 后。这十年她种种子，2034 之后开花。」

「你呢，2026 年这一年是个挺难受的年——火气浮，财来财去，想做的事很多但落地慢。这是命格自身的问题，不是你做错了什么。出口在『写下来』『记下来』『讲出来』——你印星旺，靠创作释放。」

「最关键的一句话：你们不是在比谁爆发得快，是在配合一种节奏。她快的时候，你稳。你慢的时候，她在身后给你水。这是古书里『鸾凤合鸣』的真意。」

06

2026 丙午 · 流年深读 · 为什么

+

这一年值得专门拆——因为太岁丙午对你们俩的作用截然相反，但又彼此咬合。

对她 · 用神大透 ★★★★★

她戊土日主最缺的就是丙火。流年丙午直接把这个最大缺口补上。叠加大运丁未——丁是丙的兄弟（同为火），未是比肩助身。整个 2026 年，她的命局结构在「土生金、金生水、水克火、火生土」的环上每一处都通了。

「十月戊土，气薄力微，时值小阳，先用甲木疏土，次取丙火暖土。非丙土不暖，安能发生万物。」 — 《穷通宝鉴》十月戊土

古书原话：「丙为当旺之神，真神得时得用，宜为廊庙之财」。也就是说，2026 这一年她的命格用神「真正得用」。一辈子的几次重要命运转折之一。

对你 · 比劫年 ★★★（守成 + 内压）

你的丙火日主，碰到流年丙午——同性比肩 + 劫财。表面看似帮身，实质是「火上加火、土更燥、水更难透」。古书称这种结构为「火炎土燥」：

「一派丙丁无壬癸，为火炎土燥，不得壬癸为救，僧道孤贫之命……火土燥烈，水不能存。」 — 《穷通宝鉴》四月戊土（同理论）

你不会真的「孤贫」——你大运辛卯有金生水的暗补。但「内火浮、外财乱、想做的事落不下来」是真实的体感。原因在于：

  • 大运辛卯 + 流年丙午 → 辛丙合化（你的财被合走，看似收入增加，实质是被劫）
  • 卯（你正印）与午（流年支）是「卯午破」 → 你的学习状态、专注力被打散
  • 用神壬水继续藏在申中没透 → 内在压力释放不出去

对你们俩 · 丙辛再合，命运感强

命盘原文（合盘报告）写：

「大运辛卯 × 流年丙午 → 丙辛再合水。两人「丙辛合」在此年被流年再度激活，用神水的化合力量最强。午年与命局时支午相应，婚约此年有强烈的「命运感」。」

翻译成大白话：你们俩 2026 不论结婚、订婚、合作、共同投资，都比平常年份多一份「天意」感——因为你们俩之间最神奇的"丙辛合化水"被太岁这年再次激活。

07

最近应该注意什么 · 为什么

+

命理上"注意"不等于"避免"——是「主动配合命格的运行逻辑」。下面 5 条都有古典依据。

① 申申自刑 · 守住「各做各的」边界

你日支申、她日支申。两个申放一起，按古法属「三刑」之一——"自刑"。古书原话：

「申申自刑，主配偶宫有内耗。」 — 三命通会神煞章 / 合盘原文核验通过

原因：你们俩日支同气，能量太相似——容易在一些"小事的细节"上互相纠缠（你也想做这个，她也想做这个；你想用这种方式，她也想用这种方式）。

解法：主动建立"各自的领域"——她的事业里你不指手画脚；你的创作里她给你独处时间。恰恰因为你们五行互补，反而要在"做事"上保持各自的轨道，五行才能真正发挥互补。

② 你的"财克印" · 钱进来时同步买书

你大运辛卯 + 流年丙午，财（辛）来助也来劫，印（卯）被破。古书：「辛金克乙木，财克印」——财多了，思考力反而下降。

原因：命局里你的"印星"（学习、创作、思考）和"财星"（事业、收入）天然对立。今年财动得猛，印就被压。

解法：每次有较大收入到账（无论 OpenAI 拨款 / 知识星球分红 / 项目结算），同步设一个"印星预算"——拿出 5%-10% 强制用在书、课程、创作工具、写作时间块。这是给印星"喂粮"。

③ 她的"种子期"心态 · 不要用你的尺子量她

古书原话（合盘原文）：「丁未运是种子期，丙午运是收获期。」她现在做的所有事，按命理是为 2034 后存子弹，不是当下变现。

原因：她大运丁未"印星入命"——是智慧、贵人、精神成长的时期。这种成长在外人看不出来。

解法：当她说"我在学这个 / 想转方向 / 慢一点"——你不要用你 22-32 岁壬辰运的爆发节奏来衡量她。她的爆发在 42-52 岁。你的角色是"那个稳定照射的丙火"，而不是"催她快跑的人"。

④ 共同决策窗口 · 婚 / 创业 / 大投资

合盘原文标注 2026 是第一推荐结婚年份（★★★★★），原因：

  • 对你：流年丙 + 命局时支午相应 → 强烈的命运感
  • 对她：流年丙 = 第一用神直接透干 → 一辈子级的关键年
  • 双方：丙辛合化水 + 大运辛卯 + 流年丙午 → 三重激活

但合盘也警告：避开 申月（农历七月，约 8月）办大事——叠加申申自刑加重。推荐月份：甲午月（农历五月，约 6 月初-7 月初）或壬辰月（农历三月，约 4 月）。

⑤ 你的健康 · 留意"水"系

你五行水极弱（0.8 分），这一年丙午让水更难透。古书《穷通宝鉴》对此类格局后期有警示：

「己丑运（52-62 岁）须注意健康（土克水，肾/泌尿系统）、防范财务风险。」 — 你的命盘原文

原因：不是说今年就要出问题，是命局长期"水弱"已经是一个内在配置。今年丙午加剧。

解法：多喝水（字面意义）、注意肾和泌尿系统的小信号、不要长期熬夜（夜属水，熬夜消耗你最缺的能量）。

底层一句话

她在能量爆发期，你在能量守成期。你们配合的方式不是"同频共振"，而是"轮替供能"——这一年她负责光，你负责土。
